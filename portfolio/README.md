# Portfolio

Готовые адаптации шаблона Expert Platform по нишам — скриншоты и видео для портфолио, рекламных объявлений
(Авито, Kwork и т.д.) и презентации студии.

Структура: `portfolio/<niche>/<client>/`

## Ниши

- `psychologist/` — психолог
  - `maria-polyakova/` — [Мария Полякова](./psychologist/maria-polyakova/README.md)

По мере появления новых клиентов на этом шаблоне (nutritionist, coach, astrologer и т.д.) добавляйте
новую папку `portfolio/<niche>/<client>/` с `README.md`, `screenshots/` и `video/` по тому же образцу.

Коммерческая карточка клиента (бриф, статус, доступы) ведется отдельно —
в `AI-Product-Studio/13_Clients/CLIENT-XXX_Имя/`.
