# Портфолио

Скриншоты и видео готовых сайтов по клиентам — для портфолио, рекламных объявлений
(Авито, Kwork и т.д.) и презентации студии.

Структура: `Портфолио/<Клиент>/`

## Клиенты

- [Мария Полякова](./Мария%20Полякова/README.md)

со скриншотами и видео по тому же образцу.
