# PROJECT TEMPLATE

Это шаблон проекта клиента.

Для каждого нового клиента создается копия этой папки.

Структура проекта едина для всех клиентов.

Все AI-сотрудники работают только по этой структуре.

## Структура

00_Project — информация о проекте

01_Source — все материалы, полученные от клиента

02_Website — сайт

03_Workspace — Notion Workspace

04_Online_Booking — онлайн-запись

05_Content — контент

06_AI — AI-модули

07_Delivery — финальные материалы для передачи клиенту
