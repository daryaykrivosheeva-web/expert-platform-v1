# AI Employees

В данной папке находятся AI-сотрудники проекта.

Каждый AI-сотрудник отвечает только за свой модуль.

AI-сотрудники не должны изменять работу друг друга.

Текущий список:

- Website Product Builder
- Workspace Builder (планируется)
- Content Builder (планируется)
- Booking Builder (планируется)
- Telegram Bot Builder (планируется)
