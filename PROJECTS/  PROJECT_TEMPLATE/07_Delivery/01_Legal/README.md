# Legal Documents

В данной папке хранятся юридические документы проекта.

Обязательные документы:

- Реквизиты владельца
- Политика обработки персональных данных
- Пользовательское соглашение
- Публичная оферта (при необходимости)
- Cookie Policy
- Согласие на обработку персональных данных

Все документы создаются Website Product Builder автоматически на основании данных клиента.
