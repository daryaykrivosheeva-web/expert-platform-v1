# Как добавить нового клиента

Откройте раздел "Клиенты".

Нажмите кнопку "New".

Заполните карточку.

Готово.

---

# Как добавить новую запись

Откройте раздел "Записи".

Создайте новую запись.

Свяжите ее с клиентом.

---

# Как добавить оплату

Откройте раздел "Финансы".

Создайте новую запись.

Выберите клиента.

Укажите сумму.

---

# Как добавить задачу

Перейдите в раздел "Задачи".

Создайте новую задачу.

Установите срок выполнения.

---

# Как планировать контент

Откройте раздел "Контент".

Создайте новую карточку.

Выберите тип публикации.

Назначьте дату.

---

# Рекомендации

Просматривайте Dashboard каждый день.

Это позволит видеть всю текущую работу в одном месте.
