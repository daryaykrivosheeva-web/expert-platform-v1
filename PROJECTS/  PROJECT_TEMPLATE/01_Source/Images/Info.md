# Images

В данной папке хранятся все изображения проекта.

Структура:

- hero.jpg — главное изображение первого экрана
- about.jpg — фотография для блока "Обо мне"
- cta.jpg — изображение финального блока
- gallery-01.jpg
- gallery-02.jpg
- gallery-03.jpg
- review-01.jpg
- review-02.jpg
- review-03.jpg
- favicon.png
- og-image.jpg

Правила:

- Использовать только изображения клиента.
- Если клиент не предоставил изображения, Website Product Builder может подобрать качественные бесплатные изображения (Pexels/Unsplash) для предварительной версии сайта.
- Перед финальной сдачей проекта рекомендуется заменить их на фотографии клиента.
- Все изображения должны храниться локально внутри проекта.
