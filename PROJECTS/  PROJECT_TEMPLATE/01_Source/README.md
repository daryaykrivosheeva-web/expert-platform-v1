# Source

В данной папке находятся все материалы, полученные от клиента.

Структура:

- CLIENT_DATA.yaml — вся информация о клиенте
- Images — фотографии
- Logo — логотипы
- Brand — фирменный стиль
- Documents — документы клиента

Все AI-сотрудники используют только материалы из данной папки.

Запрещено искать информацию в других папках проекта.
