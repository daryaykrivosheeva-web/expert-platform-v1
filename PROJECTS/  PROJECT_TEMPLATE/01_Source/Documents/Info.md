# Documents

В данной папке хранятся документы, предоставленные клиентом.

Например:

- Реквизиты
- Сертификаты
- Дипломы
- Лицензии
- Коммерческое предложение
- Договор
- Заполненный бриф

Website Product Builder может использовать документы данной папки для заполнения CLIENT_DATA.yaml и создания юридических документов сайта.

При недостатке информации AI не должен придумывать данные, а обязан сформировать список недостающих сведений.
